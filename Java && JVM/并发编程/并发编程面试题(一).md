# 并发编程面试题

**并发编程中数据安全的三个必备条件** 

1. 原子性 ： 是指在一个操作中就是cpu不可以在中途暂停然后再调度，既不被中断操作，要不执行完成，要不就不执行。
2. 可见性 ： 是指当多个线程访问同一个变量时，一个线程修改了这个变量的值，其他线程能够立即看得到修改的值。
3. 有序性 : 即程序执行的顺序按照代码的先后顺序执行
